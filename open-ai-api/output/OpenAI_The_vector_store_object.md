# [The vector store object Beta](/docs/api-reference/vector-stores/object)
A vector store is a collection of processed files can be used by the file_search tool.

    "in_progress": 0,
    "completed": 100,
    "cancelled": 0,
    "failed": 0,
    "total": 100

}
